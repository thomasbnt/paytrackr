# PayTrackr

> Serverless functions for PayTrackr extension

## Build Setup

```bash
# Set sendgrid keys and template id
$ cp .env.example .env

# install dependencies
$ npm install

# serve at localhost:3030
$ npm start
```

## Deployment

```bash
# Deploy to vercel
$ npm run deploy
```
